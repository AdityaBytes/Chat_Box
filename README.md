# Chat Application

This is a simple chat application built using Node.js, Express, MongoDB, and EJS.

## Features

- View all chats
- Add new chats
- Edit existing chats
- Delete chats

## Prerequisites

Before you begin, ensure you have met the following requirements:
- Node.js installed
- MongoDB installed and running
